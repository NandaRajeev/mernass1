# MERN Repository

## Description
This is a simple web project created using HTML, CSS, and JavaScript. It demonstrates the basic structure of a webpage and serves as a starting point for learning web development.

## Project Structure

```
MERN/
│── ass_1.html
│── ass_1.css
│── ass_1.js
└── README.md
```

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Features

- Simple webpage layout
- External CSS styling
- External JavaScript integration
- Beginner-friendly project structure

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/NandaRajeev/mernrepo.git
   ```

2. Open the project folder.

3. Open `ass_1.html` in your web browser.
